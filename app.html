Privacy FAQ

No. Nothing is collected at all. There are no advertising SDKs too.

CS Secure Files is highly inspired by the well-known ClamAV. While we use our own AV engine, we use parts of their virus database which is updated regularly. Thousands of new viruses appear daily, and so it is important to update the database we provide (defs.vxpack) via the in-app button. This is not compulsory but highly recommended for the best security.

Yes, you can. By law we are required to make the virus database open source since we use parts of ClamAV’s database. You can view it here.

You can install apps such as TrackerControl Slim (TC Slim). This shows which tracking SDKs your apps use.

The AV within the app is proprietary technology that is unique to ColourSwift.

Unlike many apps with both foreground and background data usage, CS Secure Files will only connect to our database servers when you click the update definitions button, causing a small temporary spike in foreground data usage. If you do not update the database for years, there will be zero data usage.

Permission Why It’s Needed
Manage External Storage Lets the file manager access all folders, required for browsing and managing files.
Storage Access / Read & Write Needed on older Android versions for reading user files and extracting archives.
Post Notifications Required on Android 13+ to show the “Protection Active” notification.
Install Packages Allows launching the native APK installer when installing apps manually.
Foreground Service Keeps real-time protection alive and shows persistent notifications.
Internet Used only to check for and download antivirus definition updates securely.
