Terminal Help

Use help at any time to open this page.

Commands

help

help

Opens the terminal documentation.

clear

clear

Clears all terminal outputs.

info

info

Displays engine version and basic device information.

scan

scan [path]

Scans a file or directory using the local antivirus engine.

hash

hash [path]

Calculates MD5 and SHA256 hashes for the selected file.

check

check [url]

Checks whether a website is associated with malicious or unsafe content.

cloud status

cloud status

Checks whether VTTI is reachable by sending a test hash.

logs on

logs on

Enables application process logging.

logs off

logs off

Disables application process logging.

proc on

proc on

Enables system process monitoring.

proc off

proc off

Disables system process monitoring.
