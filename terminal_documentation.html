Terminal help page

The CS Security terminal provides direct access to diagnostic and scanning commands.Future versions will include scripting support.

Use help at any time to open this page.

Syntax

Commands are space separated. Arguments in brackets are required unless marked optional.

command [required] <optional>
  • Commands are lowercase.
  • Paths must be valid device paths.
  • URLs should include a scheme when possible, for example https://example.com.
  • Errors return a message describing what failed.

Commands

Available terminal commands.

help

help

Displays a short command summary and opens the terminal documentation.

clear

clear

Clears all terminal output.

info

info

Displays engine version and basic device information.

scan

scan [path]

Scans a file or directory using the local antivirus engine.

scan /storage/emulated/0/Download/app.apk

hash

hash pick

Calculates MD5 and SHA256 hashes for a selected file.

hash /storage/emulated/0/Download/unknown.apk

Commands

Additional terminal commands.

check

check [url]

Checks whether a website is associated with malicious or unsafe content.

check https://example.com

cloud status

cloud status

Checks whether the cloud service is reachable by sending a test hash.

Coming Soon

These commands are planned but not yet available.

network ping

network ping [host]

Will test network reachability and latency to a domain or service.

ip ping

ip ping [address]

Will perform a low-level ping to an IP address.

exif

exif [file]

Will extract and display EXIF metadata from supported files.

Examples

Minimal command sequences.

cloud status check https://github.com scan /storage/emulated/0/Download hash /storage/emulated/0/Download/unknown.apk

Notes

  • Terminal commands do not upload files.
  • Cloud related commands only transmit hashes when required by the command.
  • Some commands may require device permissions depending on the selected path.
  • Output varies based on enabled features and local database versions.
  • The terminal is still in active developement, please forgive any bugs.
